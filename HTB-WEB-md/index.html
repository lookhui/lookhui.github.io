Malicious Input Detected

如果是的情况下就会在输出一下这个变量，查资料得到这个正则匹配可以通过匹配 %0a 来达到绕过手段也就是说使用 \n 就能绕过和 php 一样，那么直接在 index.erb 里面输入 1%0a</h1><%25=open ('|cat f*').read ()%25><h1 > 这样就能达到任意命令执行的效果
